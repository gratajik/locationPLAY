LocationAlert A generic Windows application that can use any user ID and group to alert you when you are near a location of interest (from 1 foot to 10000 feet). 

IraqiAlert!

 

Windows application that Uses the "Iraqi Weapons of Mass Destruction" list to detect if you are within 1000 feet of a Iraqi Bio, Chemical, Missle, or Nuclear site.  This AE Client application will working with any NMEA-compatible GPS attached to a serial port.  Can ALSO run in simulated mode (you can pick a site to be near; it will simulate your location at around 150 feet to that location).

Full source code is available, as a sample of how to use the AE Service.
